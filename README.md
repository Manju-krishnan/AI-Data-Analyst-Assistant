# AI-Powered Data Analyst Assistant

## 📌 Project Overview
This project is an AI-powered data analytics solution that converts natural language queries into SQL and generates insights and visualizations.

## 🚀 Features
- Natural Language to SQL
- Data Analysis using BigQuery
- Insight Generation
- Sales Trend Visualization

## 🛠️ Technologies Used
- Python
- SQL (BigQuery)
- Excel (Visualization)
- Google Cloud

## 📊 Sample Output
Includes:
- SQL query execution
- Sales trend chart
- Insights generated

Helps businesses analyze data quickly without needing technical expertise.
